## 🛠️ Tech Stack

- **Backend:** Java 11+, Spring Boot (MVC)
- **Database:** MySQL / PostgreSQL
- **ORM:** Hibernate / Spring Data JPA
- **Build Tool:** Maven
- **Frontend:** Thymeleaf / JavaFX (Web/Desktop)

---

## ✨ Key Features

- **Patient:** Book/Reschedule appointments, View Medical History, Pay Bills
- **Doctor:** Update Diagnosis, View Schedule, Manage Availability
- **Receptionist:** Generate Bills, Register Walk-ins, Verify Appointments
- **Admin:** Manage Users, Generate Reports, Monitor System
